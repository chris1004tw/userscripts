# Chris's Userscripts

## 安裝方式

1. 安裝 [Tampermonkey](https://www.tampermonkey.net/) 瀏覽器擴充套件（支援 Chrome、Firefox、Edge、Safari）
2. 點擊下方的 <kbd>Install</kbd> 連結即可安裝對應腳本

## 腳本列表

| 名稱                   | 說明                                                                                                                                                                    | 安裝                                                                                           |
| :--------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :--------------------------------------------------------------------------------------------- |
| 替換字體為 AppleGothic | 將頁面字體改為 AppleGothic（簡體用 AppleGothicSC），且還原字體替換對 Icon 的影響。支援黑名單管理，可針對特定網站停用。                                                  | [Install](https://github.com/chris1004tw/userscripts/raw/main/force-fonts-applegothic.user.js) |
| 移除 URL 追蹤          | 自動移除 URL 中的追蹤參數，保護您的隱私。支援 Google、Facebook、Twitter、YouTube、淘寶等主流網站，部分規則引用自 [ClearURLs](https://github.com/ClearURLs/Addon) 專案。 | [Install](https://github.com/chris1004tw/userscripts/raw/main/remove-url-tracker.user.js)      |
| 複製當前網址           | 按下 <kbd>Ctrl</kbd>+<kbd>Shift</kbd>+<kbd>C</kbd> 複製當前網址。在 X/Twitter 上會自動轉換為 [fxTwitter](https://github.com/FixTweet/FxTwitter) 連結並新增複製按鈕。    | [Install](https://github.com/chris1004tw/userscripts/raw/main/copy-current-url.user.js)        |
